# Captain EPM Product Context

## Product Overview
Captain EPM is an innovative Excel Add-in designed specifically for Oracle EPM Cloud administration, analytics, and automation. It unifies scattered workflows into a single, cohesive interface directly within Excel, the tool finance professionals use most.

## Core Value Proposition
- **Unified Interface:** Manage users, security, variables, and jobs without leaving Excel.
- **AI-Powered:** "Captain AI" allows users to chat with their EPM data, formulas, and metadata using natural language.
- **Security & Compliance:** Automated drift detection identifies unauthorized changes to security settings.
- **Efficiency:** Drastically reduces time spent on routine administrative tasks.

## Key Features

### 1. User & Role Management
- Provision and de-provision users across environments.
- Assign and manage roles easily.
- Audit logs for all user changes.

### 2. Dashboard Builder
- Drag-and-drop interface to create rich, interactive dashboards.
- Visualize EPM data with charts and graphs.
- Share dashboards securely with other users.

### 3. AI Copilot (Captain AI)
- Chat interface tailored for Oracle EPM.
- Ask questions like "Why is net income down in Q3?" or "Show me the trend for OpEx."
- Can explain complex formulas and member logic.

### 4. Security Drift Detection
- Automated scans to compare current security ACLs against a baseline.
- Flags unauthorized access or changes (Drift).
- One-click remediation to restore approved settings.

### 5. Substitution Variables Manager
- View, edit, and audit substitution variables across multiple applications.
- visual interface for managing complex variable sets.

### 6. Job Console & Automation
- Launch, monitor, and schedule EPM jobs (like Cubes Refresh, Data Import/Export).
- Detailed logs and error reporting.
